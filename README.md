# Student Result Management System (Python)

A console-based Python application to manage and calculate student results.

## Features
- Enter student name and marks
- Calculate total and percentage
- Assign grade
- Display result

## Concepts Used
- Input / Output
- Variables
- Conditional Statements
- Loops
- Functions

## How to Run
1. Make sure Python is installed
2. Run the command:
   python main.py
